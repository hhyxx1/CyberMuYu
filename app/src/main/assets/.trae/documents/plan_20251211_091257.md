# 新增特殊功能实现计划

## 功能1：连击特效系统
- **触发条件**：当连续点击次数达到10、25、50、100时
- **视觉效果**：不同级别的粒子爆发、颜色变化、特殊动画
- **音效效果**：连击音效叠加、音调变化
- **实现方式**：在现有连击计数基础上添加特效触发逻辑

## 功能2：成就系统
- **成就类型**：
  - 首次连击10次
  - 累计点击1000次
  - 单日点击500次
  - 体验所有主题
  - 体验所有音效
  - 使用白噪音1小时
- **存储方式**：localStorage持久化存储
- **展示方式**：成就弹窗提示、成就列表查看

## 功能3：自定义木鱼外观
- **新增木鱼样式**：赛博、传统、水晶、金属4种样式
- **实现方式**：通过CSS类切换不同样式，添加选择器UI

## 功能4：冥想模式
- **冥想时长**：5分钟、10分钟、15分钟可选
- **功能特点**：
  - 自动播放引导音效
  - 渐变背景效果
  - 时间倒计时
  - 结束提醒

## 功能5：振动反馈（移动端）
- **触发条件**：点击木鱼时
- **实现方式**：使用Web Vibration API
- **适配**：仅在支持振动的设备上启用

## 文件修改计划
1. **index.html**：添加新功能的UI元素
2. **script.js**：实现新功能的核心逻辑
3. **style.css**：添加新功能的样式
4. **manifest.json**：更新应用描述和功能标签

## 实现步骤
1. 首先实现连击特效系统
2. 然后添加成就系统
3. 接着实现自定义木鱼外观
4. 最后添加冥想模式和振动反馈

所有功能将保持现有UI风格，确保良好的用户体验和响应式设计。
