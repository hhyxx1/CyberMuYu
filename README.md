# CyberMuYu - 赛博风格电子木鱼

## 项目概述

CyberMuYu 是一个基于 Web 技术和 Android 平台开发的赛博风格电子木鱼应用，结合了现代 UI 设计与传统木鱼功能，旨在为用户提供一个解压、冥想和放松的工具。这是一个人工智能选修课项目，使用 AI 辅助开发完成。

### 开发背景

随着现代生活节奏的加快，人们面临着越来越大的压力。电子木鱼作为一种简单易用的解压工具，受到了广泛欢迎。本项目结合了赛博朋克美学与传统木鱼功能，通过 AI 辅助开发，实现了一个功能丰富、界面美观的电子木鱼应用。

### 主要功能

- ✨ **赛博风格设计**：炫酷的渐变背景、发光效果和动态粒子
- 🎵 **多种音效选择**：木鱼、铃铛、水滴、风铃
- 🎨 **主题切换**：赛博朋克、柔和粉、深海蓝、日落橙
- 🐟 **多种木鱼样式**：赛博、传统、水晶、金属、火焰、寒冰
- 🌊 **白噪音功能**：支持自定义音乐上传
- 🧘 **呼吸引导**：帮助用户进行深呼吸放松
- 📱 **冥想模式**：支持自定义冥想时长
- 🏆 **成就系统**：记录用户使用情况
- 🌐 **响应式设计**：适配不同设备尺寸
- 📱 **Android 应用**：支持 APK 安装

## 技术栈

### 前端技术
- **HTML5**：页面结构
- **CSS3**：样式设计，包括渐变、动画、响应式布局
- **JavaScript (ES6+)**：交互逻辑，音频处理，设备传感器交互

### Android 技术
- **Java**：Android 应用开发
- **WebView**：嵌入 Web 应用
- **MediaSession API**：系统媒体控制
- **JavaScriptInterface**：Web 与 Android 通信

### AI 辅助工具
- **AI 代码生成**：生成基础代码结构和功能模块
- **AI 设计建议**：UI/UX 设计优化
- **AI 调试辅助**：定位和修复代码错误
- **AI 文档生成**：自动生成技术文档

## 设计思路

### 核心设计理念
- **传统与现代结合**：将传统木鱼功能与赛博朋克美学结合
- **用户体验优先**：简洁直观的界面设计
- **性能优化**：确保应用流畅运行
- **可扩展性**：便于添加新功能和主题

### 架构设计
- **前后端分离**：Web 应用负责 UI 和交互，Android 负责平台集成
- **模块化设计**：功能模块独立，便于维护和扩展
- **响应式布局**：适配不同设备尺寸
